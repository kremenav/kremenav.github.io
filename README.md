# Personal Website

Source code for my personal website.

Live site: https://kremenav.github.io

## Built With

* [Hugo](https://gohugo.io/) (static site generator)
* Hugo Blox / Academic-style components
* Hosted on GitHub Pages

## Local Development

Install Hugo and run:

```
hugo server
```

Then open:

```
http://localhost:1313
```

## Deployment

The site is automatically built and deployed via GitHub Pages when changes are pushed to the repository.

## Analytics

This site uses GoatCounter for privacy-friendly analytics.
